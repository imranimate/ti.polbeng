Laboratorium

Laboratorium Sistem Informasi dan Basis Data

Digunakan untuk Praktikum : Basis Data, Perancangan Basis Data, Administrasi Basis Data, Teori dan Praktikum Konsep Pemogram 1,2, Pekerjaan tugas mata kuliah, tugas akhir (TA), dan pengabdian masyarakat.

Laboratorium Jaringan Komputer dan Informasi

Digunakan untuk praktikum : Sistem Operasi, Konsep Jaringan, Administrasi Sistem, Administrasi Jaringan, Sistem informasi manajemen, Pelatihan Cisco Local Academy, Pekerjaan tugas mata kuliah, tugas akhir (TA), komputasi berbasis jaringan, dan Pengabdian Pada Masyarakat.

Laboratorium Pembelajaran Multimedia

Dapat digunakan untuk Praktikum : Animasi Multimedia, Desain Grafis,Video Editing, MYOB, Pekerjaan tugas mata kuliah, tugas akhir (TA), dan pengabdian masyarakat.

Laboratorium Pemrograman Tingkat Lanjut

Digunakan untuk praktikum : Pemograman Berorientasi Objek (PBO), Struktur Data, Pekerjaan tugas mata kuliah, Pelatihan Java Fundamental (Oracle Academy) tugas akhir (TA), dan pengabdian masyarakat).

Laboratorium Sistem Cerdas

Digunakan untuk Praktikum: Pemrograman, Desain Web, Web Programming, Pekerjaan tugas mata kuliah, tugas akhir (TA), dan Pengabdian Pada Masyarakat.

Laboratorium Komputer Dasar

Digunakan untuk Praktikum : Pengantar Teknologi Informasi, Perawatan dan Perbaikan Komputer, tugas akhir (TA), dan pengabdian masyarakat.

Laboratorium Rekayasa Perangkat Lunak

Digunakan untuk praktikum : Pengantar Rekayasa Perangkat Lunak, Aplikasi Web dan Mobile.

Laboratorium Komputasi

Digunakan untuk praktikum : Pemrograman I, Pemrograman Web II, Teknologi Web XML dan Web Service, Pekerjaan tugas mata kuliah, tugas akhir (TA), dan pengabdian masyarakat.
